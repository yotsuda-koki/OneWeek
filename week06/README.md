# OrbitSim

**OrbitSim** は、地球が太陽の重力によって楕円軌道を描く様子をシンプルな数値計算でアニメーション表示する Python プログラムです。

天体物理の基本法則（万有引力とニュートン運動の法則）に基づき、地球の軌道運動を毎時刻更新しながら描画します。

---

## 主な機能

- 太陽と地球の2体問題をシンプルに再現
- matplotlib アニメーションによる軌道描画
- 距離・速度・加速度の更新を毎時刻ステップで計算
- 実際の質量・距離・速度（SI単位）を使用した現実的スケール
- 地球の軌道と移動の軌跡を可視化

---

## 実行方法

### 1. 必要なパッケージのインストール

Python 3 がインストールされた環境で、以下のライブラリをインストールしてください：

```
pip install matplotlib numpy
```

### 2. `solar_orbit_sim.py` を実行

```
python solar_orbit_sim.py
```

実行すると、アニメーションウィンドウが開き、地球の公転軌道が可視化されます。

---

## 計算パラメータ

| 項目         | 値                 |
|--------------|--------------------|
| 太陽質量     | `1.989 × 10³⁰ kg` |
| 地球質量     | `5.972 × 10²⁴ kg` |
| 地球初期位置 | `1.496 × 10¹¹ m` （約1AU）|
| 地球速度     | `29780 m/s`（円軌道を近似）|
| 時間刻み     | `3600秒`（1時間）|
| シミュレーション期間 | 1年分（365×24時間）|

---

## 物理法則

- 万有引力の法則：
  ```
  F = G * M * m / r²
  ```

- 運動方程式（ニュートンの第2法則）：
  ```
  F = m * a
  ```

加速度は、重力によって常に中心（太陽）に向かって作用し、それを地球の速度ベクトルに加算しながら、位置を更新しています。

---

## 補足

- 実際の太陽系には他の惑星や非線形な摂動がありますが、このシミュレーションでは **2体問題**（太陽と地球）に限定しています。
- 実行中は `matplotlib` のアニメーションが動的に更新されます。
