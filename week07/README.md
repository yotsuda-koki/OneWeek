# FallSim

**FallSim** は、異なる素材の物体が地面に落下する様子を、空気抵抗を含めてシミュレーションするアプリです。  
Pygame による視覚表示と Tkinter ベースのGUI入力により、簡単に物理的落下の違いを体験できます。

---

## 主な機能

- 素材を2つ選んで比較落下
- 各素材に「質量・断面積・抗力係数」を設定可能
- プリセット素材（羽毛、石、木、鉄球、質点）を内蔵
- 「カスタム素材」は自由入力＋名前入力可能
- 高さや重力加速度を自由に変更可能
- 落下中の速度と着地時間をリアルタイムに表示
- Pygameでの落下アニメーション表示（時間と距離のメモリ付き）
- 落下完了後も画面は閉じず、`Esc`キーまたはウィンドウ操作で終了可能

---

## 使い方

1. Python 3.x をインストール（例：3.10〜3.13）
2. 以下のライブラリをインストール：

```
pip install pygame
```

3. `fallSim.py` を実行：

```
python fallSim.py
```

4. GUIが起動したら、素材と条件を選び、「シミュレーション開始」ボタンを押す
