# JPDSL

**日本語DSLプログラミングアプリ**は、Python + Streamlit を使って、  
**日本語だけでプログラムが書ける**初心者・教育向けの軽量Webアプリです。  
直感的な日本語文で、変数の代入・計算・条件分岐・繰り返しがすぐに実行できます。

---

## 主な機能

- 日本語の文章で簡単にプログラムを記述
- DSL構文をPythonコードにリアルタイム変換＆実行
- 条件分岐やくりかえし、表示も日本語だけで記述可能
- Webブラウザで即利用（Streamlit UI）

---

## 使用例

| 入力例（日本語DSL）                              | 実行内容（Python相当）                        |
|--------------------------------------------------|-----------------------------------------------|
| `x を 3 にする`                                   | `x = 3`                                       |
| `x を x + 1 にする`                               | `x = x + 1`                                   |
| `x を表示する`                                    | `print(x)`                                    |
| `「こんにちは」と表示する`                        | `print("こんにちは")`                          |
| `x が 5 より大きいなら「大きい」と表示する`       | `if x > 5: print("大きい")`                   |
| `3 回くりかえす：「こんにちは」と表示する`        | `for _ in range(3): print("こんにちは")`       |
| `x が偶数なら「偶数です」と表示する`              | `if x % 2 == 0: print("偶数です")`            |
| `もし x が 5 より大きいなら「OK」そうでなければ「NG」と表示する` | `if x > 5: print("OK") else: print("NG")` |

## ファイル構成

| ファイル名       | 説明                                |
|------------------|-------------------------------------|
| `app.py`         | Streamlit製UIアプリ本体              |
| `dsl_runner.py`  | 日本語DSL文をPythonコードへ変換するロジック |

---

## 使用方法

1. このリポジトリをクローン or ZIPでダウンロード
2. 必要なパッケージをインストール：

```bash
pip install streamlit
```

3. アプリを起動：

```bash
streamlit run app.py
```

4. ブラウザが自動で開き、`http://localhost:8501` にアクセスできます。

---

## 注意点

- 現在は **1行に1命令** のみ対応（ネストや複数行の繰り返しは未対応）
- 表示文は必ず **全角の「〜」** で囲んでください
- サーバー側のインストールが必要です（完全クライアント動作ではありません）
