
# Simulacra Corp. - ダミー企業サイト

**Simulacra Corp.** は、現代的な外資系企業のように見える、架空のダミー企業ウェブサイトです。ポートフォリオやデモ用途を目的としており、企業サイトに見られる典型的な機能（お問い合わせフォーム、会社概要、管理画面など）を備えています。

---

## 主な機能

- PHPベースのダミー企業サイト
- Bootstrap を用いたクリーンでモダンなレイアウト
- モジュール構造（`header.php`, `footer.php`）で保守性向上
- バリデーション付きお問い合わせフォーム
- 管理者用パネルで送信メッセージを閲覧可能
- お問い合わせデータをCSVでエクスポート可能
- エラーページ対応（`403.php`, `500.php`）
- 軽量でローカルでも簡単に動作

---

## 使用技術

- PHP 8以上
- HTML / CSS（Bootstrap利用）
- JavaScript（基本的なバリデーション）
- SQLite または MySQL（PDO経由で互換性あり）
- PHPMailer（メール送信機能）
- Composer（依存管理）

---

## ファイル構成

- `index.php` – トップページ
- `about.php` – 会社概要ページ
- `contact.php` – お問い合わせフォーム
- `admin.php` – 管理画面（シンプルな認証）
- `export.php` – CSVエクスポート機能
- `config.php` – データベースとメール設定
- `auth.php` – 管理者認証処理
- `includes/` – ヘッダー・フッターなど共通パーツ
- `assets/` – 静的アセット（CSS, 画像, Faviconなど）
- `vendor/` – Composerライブラリ群

---

## 実行方法

1. PHP（バージョン8.0以上）をインストール
2. このリポジトリをクローンまたはダウンロード
3. XAMPPなどのローカルサーバー環境の `htdocs` に配置
4. `database.sql` を使用してデータベースを作成
5. `http://localhost/homepage/` にアクセスして動作確認

### オプション：メール送信機能

- `config.php` にSMTP情報を設定
- ComposerでPHPMailerをインストール：

```
composer install
```

---

## お問い合わせフォームについて

送信されたメッセージはデータベースに保存されます。管理画面からすべてのメッセージを閲覧でき、`export.php` によりCSV形式でダウンロードできます。
